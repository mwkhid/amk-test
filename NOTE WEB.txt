Deskripsi Website

1. Website memiliki dua jenis akun, yaitu akun dengan role 'ADMIN' dan role 'STAFF'. 
   Role bisa dipilih ketika melakukan registrasi akun.

2. Website memiliki beberapa pilihan menu:
   a. Dashboard 	: Memuat ringkasan jumlah customer, jenis item, jumlah pembelian, dan total pembelian
   b. Customer  	: Memuat daftar dan detail data customer
   c. Item		: Memuat daftar dan detail data item
   d. Pembelian 	: Memuat daftar dan detail data pembelian
   e. User Management 	: Meemuat daftar dan detail user yang telah terdaftar dan memiliki akses ke website

3. Untuk role 'ADMIN' memiliki beberapa pilihan menu:
   a. Dashboard
   b. Customer
   c. Item
   d. Pembelian
   e. User Management
   Selain itu ADMIN juga dapat melakukan tambah data, update data dan hapus data pada semua menu.

4. Untuk role 'STAFF' hanya memiliki pilihan menu:
   a. Dashboard
   b. Customer
   c. Item
   d. Pembelian
   Selain itu pada STAFF dapat melihat data pada semua menu dan hanya bisa melakukan tambah data pada menu 
   pembelian.

5. Website menggunakan framework laravel v.10 dan untuk database menggunakan MySQL

Link github project: https://github.com/mwkhid/amk-test.git

Jika ingin import database yang telah dibuat, bisa melakukan import pada file amk_crud.sql. Akun ADMIN
